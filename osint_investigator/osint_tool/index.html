Neler Yapabilir?

🌐
Google Dorking

12 Platform Taraması
Twitter/X, Instagram, LinkedIn, GitHub, Reddit, TikTok, Pinterest, Facebook, YouTube, Twitch, Medium, Telegram.

Username Varyantları

// INSTALLATION

Nasıl Kurulur?

01 / KOLAY
Windows .exe
Releases sayfasından .exe indir, çalıştır. Python gerekmez.
⬇ Releases →
02 / GELİŞTİRİCİ
Python ile Çalıştır
git clone https://github.com/
AltayCelikkanat/osint-investigator
cd osint-investigator
pip install -r requirements.txt
python osint_gui.py
03 / WINDOWS
Otomatik Başlatıcı
OSINT_BASLAT.bat dosyasına çift tıkla. Python yoksa otomatik kurar, ardından programı başlatır.
OSINT_BASLAT.bat
# çift tıkla — bitti.
